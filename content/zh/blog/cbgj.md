| 出版广角      |               |               |               |               |               |
| ------------: | :------------ | ------------: | :-----------  | ------------: | :------------ |
| 国际刊号      | 1006-7000     | 主管单位      | 广西师范大学  | 主办单位      | 广西期刊传媒集团有限公司 |

> 《出版广角》杂志创刊于1995年，由广西师范大学主管、广西期刊传媒集团有限公司主办的一本学术期刊，目前已被CSSCI南大期刊(含扩展版)、北大期刊(中国人文社会科学期刊)等知名数据库收录，是广西师范大学主管的国家重点学术期刊之一。出版广角以“与中国出版同步，为中国出版服务”为宗旨，站在中国出版主流文化的层面，悉心关注出版文化的大事和大势，已发展成为具有理论先导地位和新锐气象的高品位专业期刊。

#### 出版广角主要栏目
> 观点、专题、实务、阅读_书评、产业、新观察_广角视阈、新观察、新传媒等

#### 出版广角获奖情况
> 2002年12月被评为第二届国家期刊奖重点社科期刊。2004年、2008年被连续评为第四届和第五届“全国新闻核心期刊”。2008年度“复印报刊资料”系列学术期刊全文转载量（率）排名第二、2009年度“复印报刊资料”系列学术期刊全文转载量（率）排名第四（据《光明日报》）。
